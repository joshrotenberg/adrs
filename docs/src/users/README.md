# User Guide

Welcome to the `adrs` user guide. This section covers everything you need to install, configure, and use the `adrs` command-line tool.

## Quick Start

```sh
# Initialize a new ADR repository
adrs init

# Create your first decision
adrs new "Use PostgreSQL for persistence"

# List all ADRs
adrs list

# Check repository health
adrs doctor
```
