## [0.7.3] - 2024-XX-XX

### Fixed
- Open actual ADR file in editor instead of temp file

## [0.7.2] - 2024-XX-XX

### Fixed
- Documentation improvements

## [0.7.0] - 2024-XX-XX

### Added
- MCP server for AI agent integration
- 15 MCP tools for ADR management
- HTTP transport support (optional)

### Changed
- Improved error messages
- Better configuration discovery

## [0.6.0] - 2024-XX-XX

### Added
- MADR 4.0.0 format support
- Template variants (full, minimal, bare)
- Tags support in NextGen mode
- JSON-ADR import/export

### Changed
- Restructured as Cargo workspace
- Library-first architecture

## [0.5.0] - 2024-XX-XX

### Added
- NextGen mode with YAML frontmatter
- Health checks (`doctor` command)
- Full-text search
