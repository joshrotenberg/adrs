# Roadmap

This roadmap outlines planned features and improvements for `adrs`.

## Current Version: 0.7.x

### Completed
- adr-tools compatibility
- MADR 4.0.0 support
- Template variants
- MCP server
- JSON-ADR import/export
- Repository health checks

## Short-term (0.8.x)

### Documentation
- [ ] Consolidate and expand documentation
- [ ] Add ADR writing guide
- [ ] Improve cross-references

### Quality
- [ ] Expand test coverage
- [ ] Add property-based tests
- [ ] Improve error messages

## Medium-term (0.9.x)

### Features
- [ ] Runtime template creation
- [ ] Additional built-in templates (RFC, SDD)
- [ ] gitconfig integration
- [ ] Man page generation

### Testing
- [ ] Component test infrastructure
- [ ] MCP server integration tests
- [ ] Parameterized test cases

## Long-term (1.0+)

### Architecture
- [ ] ADR scoping (org/team/project levels)
- [ ] Frontmatter schema validation
- [ ] External resource linking

### Storage
- [ ] Pluggable storage backends
- [ ] SQLite backend for queries
- [ ] Graph query support

### Search
- [ ] Semantic search
- [ ] Embedding support
- [ ] Advanced filtering

### Integrations
- [ ] mdbook-adrs preprocessor
- [ ] IDE plugins
- [ ] CI/CD integration guides
