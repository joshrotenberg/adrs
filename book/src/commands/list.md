# list

## Overview

`list` lists the current ADRs found in the configured (or default) directory.

## Help

```sh
List Architectural Decision Records

Usage: adrs list

Options:
  -h, --help     Print help
  -V, --version  Print version
```

## Examples

```sh
adrs list
```
